# The Notion Collective Website

A static site for The Notion Collective, an art collective that is currently inactive. The site serves as a historical archive of past projects.

## Guidelines

- Use semantic HTML (`<article>`, `<section>`, `<header>`, `<nav>`, `<time>`, etc.) — no `<div>` soup
- Keep things simple: static HTML and CSS, no JavaScript unless truly necessary
- Typography-first design — let type, whitespace, and hierarchy do the work
- The tone is archival, not promotional
- "League Gothic" is reserved for the name "The Notion Collective" — do not use it for headings, buttons, or other UI elements. The name should not be in all-caps (no `text-transform: uppercase`)
